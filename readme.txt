== Description ==
Automatically generated Woocommerce category menu. You can use menu with shortcode.

Any questions and suggestions send to: [e-mail](mailto:info@inmysight.ru) or [Telegram](https://t.me/nickbtraveler)

= Features include =
* Dynamically made menu with your Woocommerce categories
* Some parameters to change results
* Images for categories from admin page
* Mobile friendly
* Just a few requests to the database
* Translation ready (Russian by default)
* Beta brands menu

= Requires =
* Woocommerce 5 or higher

== Installation ==
1. Upload NBTech Woomenu to /wp-content/plugins/
2. Activate the Plugin via the plugins menu
3. Copy shortcode from admin page "NBTech Woomenu" anywhere you want
4. PROFIT!

== Frequently Asked Questions ==
= Which Shortcode should I use? =
Shortcode is - [NBTECH_WOOMENU]

= How can I style the menu? =
For that moment it is possible with your theme additional CSS or with creating of child theme and styling with style.css of the child theme. 
And the third way is to style menu with NBTech Woomenu plugin's CSS file, but it will be overwritten with the next update.

= How can I tune the menu? =
You can change settings of the menu with parameters. You can find all available parameters on Wordpress admin console -> NBTech Woomenu.
